# TODO App

  


This is a simple CRUD operations project made with Next.js and JSON server. 
To start the project clone this repository;
Start the JSON-Server;
Run the project;

    //new terminal
    cd todo-app
    npm i
    npm run json-server
    
    //new terminal
    cd todo-app
    npm run dev
    
JSON Server endpoint is http://localhost:3001/tasks
